# 🤖 SMMPilot — Автопостинг в Telegram

Полнофункциональный бот для автоматических публикаций в группы и каналы Telegram с панелью администратора, расписанием, Spintax и статистикой.

---

## 📋 Возможности

| Функция | Описание |
|---------|----------|
| **Создание кампаний** | Текст + фото/видео для публикаций |
| **Расписание** | Публикации по дням недели и точному времени |
| **Spintax** | Уникализация текста `{вариант1\|вариант2}` |
| **Статистика** | Отчёты по каждой кампании и группе |
| **Управление группами** | Добавление/удаление целевых групп |
| **Админ-панель** | Полный контроль для администраторов |
| **Рассылка** | Массовая отправка сообщений всем пользователям |

## 🚀 Быстрый старт

### 1. Создайте бота в Telegram

1. Напишите [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Скопируйте **API Token**

### 2. Настройте Upstash Redis

1. Зарегистрируйтесь на [console.upstash.com](https://console.upstash.com)
2. Создайте новую базу данных Redis
3. Скопируйте **Redis URL** (формат: `rediss://default:PASSWORD@HOST:PORT`)

### 3. Заполните переменные окружения

Скопируйте `.env.example` в `.env` и заполните:

```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_IDS=123456789,987654321
REDIS_URL=rediss://default:PASSWORD@HOST:PORT
WEBHOOK_URL=https://your-service.onrender.com
PORT=10000
```

> **ADMIN_IDS** — ваш Telegram ID. Узнать можно через [@userinfobot](https://t.me/userinfobot)

## 🏗 Деплой на Render

### Через GitHub (рекомендуется)

1. Создайте репозиторий на GitHub и загрузите файлы
2. Зарегистрируйтесь на [render.com](https://render.com)
3. Нажмите **New → Web Service**
4. Подключите свой GitHub-репозиторий
5. Выберите **Runtime: Python**
6. В **Environment Variables** добавьте все переменные из `.env`
7. Нажмите **Deploy**

### Через Render Blueprint

Файл `render.yaml` уже настроен. Просто:

1. Загрузите проект на GitHub
2. На Render нажмите **New → Blueprint**
3. Выберите репозиторий

## 🔄 Как работает бот

### Создание кампании
1. Нажмите **"🚀 Создать кампанию"** или `/newcampaign`
2. Введите название
3. Введите текст поста
4. Отправьте фото/видео или напишите "пропустить"
5. Включите/выключите Spintax

### Настройка расписания
1. Выберите кампанию → **"📅 Добавить расписание"**
2. Выберите группы для публикации
3. Выберите дни недели
4. Введите время (например, `14:30`)

### Spintax
Используйте `{вариант1|вариант2|вариант3}` в тексте кампании. Бот случайно выберет один вариант при каждой публикации.

Пример:
```
{Привет|Здравствуй|Добрый день}, {друзья|подписчики|все}!
Сегодня у нас {акция|спецпредложение|скидка}!
```

## 🔧 Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Бот не отвечает | Проверьте `BOT_TOKEN` |
| Ошибка Redis | Проверьте `REDIS_URL` и SSL |
| Посты не отправляются | Убедитесь, что бот админ в группе |
| Webhook не работает | Проверьте `WEBHOOK_URL` и порт |
